DayFragment.java：对应天统计界面，处理相关逻辑，获取统计数据、渲染统计图
WeekFragment.java：对应周统计界面，处理相关逻辑，获取统计数据、渲染统计图
MonthFragment.java：对应月统计界面，处理相关逻辑，获取统计数据、渲染统计图
PersonalFragment.java：对应个人界面，处理相关逻辑，显示用户基本信息以及设置自动任务的开启关闭，如自动开启勿扰模式等
ScheduleFragment.java：对应日程管理界面，处理相关逻辑，显示日程安排
StatisticsFragment.java：承载天、周、月三个统计界面，使三个界面可相互切换
